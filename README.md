# Expense Tracker MCP Server

A simple MCP (Model Context Protocol) server built with Python for learning how MCP servers work.

## Features

* Add expenses
* View expense summary
* Store expenses in a local SQLite database
* Expose functionality as MCP tools

## Tech Stack

* Python
* FastMCP
* SQLite

## Run the Server

```bash
uv sync
uv run main.py
```

## Purpose

This project was created as a beginner-friendly learning project to understand how to build and use MCP servers.
